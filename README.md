# Denoise_nft

**Code to support assumptions for generating proof of concept of [Denoise company](https://denoisenft.com/).**

![logo_image](imgs/logo.png)

## Denoise general assumptions

Social hype influences the NFT price variation.

### V0 concept

Miscellaneous code and model to explore various social platforms (Twitter, Google Trends, Discord, Twitter, ...)

### V1 concept

Export into a dashboard social metric for Upcoming and Top 100 Ranking collection

### V2 concept

Generate a Denoise Metric to track social hype for Upcoming and Top 100 Ranking collection
